# Retail_Sales_Analysis
Retail sales data analysis using Python to uncover sales trends, profitability, and loss-making segments.
Overview:
A data analysis project that uses Python to analyze the sales performance of a retail business. The project aims to analyze sales trends, profitability, and loss-making product segments.

Key Objectives:
Analyze sales and profit
Identify top-selling categories and regions
Identify loss-making sub-categories
Visualize results for decision-making

Dataset:
Public Sample Superstore retail dataset
~10,000 transaction records

Key fields: Category, Sub-Category, Sales, Profit, Region

Tools & Skills:
Python (pandas, matplotlib)
Jupyter Notebook

SQL Concepts: SELECT, GROUP BY, WHERE, HAVING, ORDER BY, SUM, AVG

Key Insights:
Technology is the highest revenue and profit-generating category
West region contributes the highest sales
Furniture shows high sales but low profit
Tables and Bookcases are major loss-making sub-categories
